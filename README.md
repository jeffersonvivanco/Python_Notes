netcat
======
To run it first run the server by doing this

- ./bhnet.py -l -p 9999 -c

**Note:** -c is to run the command shell

Then to run the client do

- ./bhnet.py -t localhost -p 9999

**Note:** Press CTRL-D to activate shell
